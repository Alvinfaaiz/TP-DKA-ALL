Pengenalan Matplotlib¶

Matplotlib merupakan library Python yang digunakan untuk membuat visualisasi data dalam bentuk grafik. Library ini menyediakan berbagai alat untuk membuat grafik seperti linechart, barchart, histogram, scatter plot, dan banyak lagi.

In [3]:
import matplotlib.pyplot as plt # Import library matplotlib.pyplot

Soal 1: Visualisasi Scatter Plot¶

Sebuah pusat perbelanjaan ingin menganalisis pola penjualan dari dua toko selama satu minggu. Data berikut menunjukkan total transaksi harian (dalam ribuan rupiah) yang terjadi di dua toko tersebut.

Gunakan scatter plot untuk memvisualisasikan hubungan antara hari ke- dan total transaksi dari kedua toko.

Hari ke- Penjualan Toko A (ribu rupiah) Penjualan Toko B (ribu rupiah)
1 200 190
2 220 210
3 200 230
4 220 260
5 270 245
6 320 310
7 290 300
In [6]:
# Data pelanggan dan total penjualan (dalam ribuan rupiah)
hari = [1, 2, 3, 4, 5, 6, 7]
penjualan_toko_a = [200, 220, 200, 220, 270, 320, 290]
penjualan_toko_b = [190, 210, 230, 260, 245, 310, 300]

# Inisialisasi figure
fig = plt.figure(figsize=(10, 5))

# Plot scatterplot untuk data Toko A
plt.scatter(hari, penjualan_toko_a, c='red', alpha=0.65, label="Toko A")

# Plot scatterplot untuk data Toko B
plt.scatter(hari, penjualan_toko_b, c='blue', alpha=0.50, label="Toko B")

# Menambahkan judul dan label
plt.title("Hubungan Jumlah Pelanggan dan Total Penjualan di Dua Toko")
plt.xlabel("Hari ke-")
plt.ylabel("Total Penjualan (Ribu Rupiah)")

# Menambahkan legend
plt.legend()

# Tampilkan scatterplot
plt.show()

In [13]:
# Data jumlah kendaraan di 5 ruas jalan
ruas_jalan = ["Jalan A", "Jalan B", "Jalan C", "Jalan D", "Jalan E"]
jumlah_kendaraan = [1100, 900, 1500, 1200, 1600]

# Inisialisasi figure
fig, ax= plt.subplots(figsize=(8, 6))

# List warna
list_color = ['red', 'blue', 'green', 'purple', 'orange']

# Plot barchart untuk data kendaraan
ax.bar(ruas_jalan, jumlah_kendaraan, color=list_color)

# Menambahkan judul dan label
ax.set_title('Jumlah Kendaraan yang Melewati 5 Ruas Jalan Utama dalam Sehari')
ax.set_xlabel('Ruas Jalan')
ax.set_ylabel('Jumlah Kendaraan')

# Tampilkan barchart
plt.show()

Soal 5: Visualisasi Boxplot¶

Sebuah layanan pengiriman makanan ingin menganalisis distribusi waktu penyelesaian pesanan oleh kurir dari tiga cabang restoran. Data berikut menunjukkan waktu penyelesaian pesanan (dalam menit) dari masing-masing cabang.

Gunakan boxplot untuk memvisualisasikan distribusi waktu penyelesaian pesanan oleh kurir.

Cabang Restoran Waktu Penyelesaian (Menit)
Cabang A 25, 30, 28, 35, 32, 31, 27, 29, 34, 33
Cabang B 20, 22, 25, 21, 23, 26, 24, 22, 27, 28
Cabang C 30, 35, 32, 31, 33, 36, 34, 30, 29, 37
In [20]:
# sesuaikan data
cabang_a = [25, 30, 28, 35, 32, 31, 27, 29, 34, 33]
cabang_b = [20, 22, 25, 21, 23, 26, 24, 22, 27, 28]
cabang_c = [30, 35, 32, 31, 33, 36, 34, 30, 29, 37]

# Gabungkan data
data_nilai = [cabang_a, cabang_b, cabang_c]

# Boxplot
plt.figure(figsize=(8,6))
plt.boxplot(data_nilai, label=["Cabang A", "Cabang B", "Cabang C"], patch_artist=True)

# Tambahkan detail
plt.xlabel("Cabang")
plt.ylabel("Nilai Ujian")
plt.title("Distribusi Nilai Ujian per Cabang")
plt.grid(True)

# Tampilkan boxplot
plt.show()
